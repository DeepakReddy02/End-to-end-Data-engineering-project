# Rainfall Data Pipeline

This folder defines all source code for the 'Rainfall Data Pipeline' pipeline:

- `Extraction`: Extracted rainfall data from the Rainfall API for Future Tranformations.
- `transformations`: Transformed raw rainfall API data by cleaning, normalizing, and structuring it for downstream us.Converted raw API responses into analytics-ready tables using transformation logic.
- `Exploration`: Performed exploratory analysis on transformed rainfall data to assess quality, trends, and anomalies.Conducted data exploration to validate transformations and support downstream analytics.
